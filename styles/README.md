# Styles

Any auxilary, style related code goes here

## fonts.ts

This is a good place to import Google ( or other ) fonts for use within your webapp. Two examples for Google fonts have been provided

## typescales.css

This file contains a few starter Typescale Systems to make your app's typography look and feel more cohesive. You can copy one system into your `globals.css` file (into the `@layer base`).
